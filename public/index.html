FAQ

What are HTML entities?

HTML entities are special codes used to represent reserved characters in HTML. They ensure that characters like <, >, and & display correctly instead of being interpreted as HTML tags.

Why do I need to encode text?

Encoding is essential for preventing XSS (Cross-Site Scripting) attacks and ensuring user-generated content displays safely on web pages without breaking the HTML structure.

What is the difference between named and numeric entities?

Named entities use memorable names like &amp;, while numeric entities use character codes like &#38; (decimal) or &#x26; (hexadecimal). All represent the same character.
